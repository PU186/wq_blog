## 🔧 信号发生器

1. **Rate** 档位是用来调节频率的
2. 信号不变化 → 原因：信号接口松动

## 🔦 光电传感器

1. 确保光电传感器收光孔**不要被物品遮挡**
2. 示波器仅有一个波形、波形无变化时 → 光电传感器被遮挡，或透光光纤与延时板光电管未对准

## ⚡ 能量计

1. 功率计探头**不是背景**，是小白板（注意区分）

## 🌈 爱万提斯光谱仪

1. 当光谱仪显示变成**连续模式**时，大概是光谱仪外触发线与延时板连接的地方接触**不良**，稍微碰一下就好
2. 四通道光谱仪需要同步各光纤，即点击 **Syanced**；需要设置积分时间为 **1.05ms**
3. 信号无变化，但光谱仪软件有变化 → 光触发未对准

## 💻 爱万提斯光谱软件

1. **LIBS** 设置时不需要设置模式，选择**外触发模式**，选项如下图一样勾选，可以连续击打

![光谱软件触发模式设置]({attach}lab-images/P1.png)
## 🔬 实验样机

1. **阶梯型脉冲**：首尾先连，原因：阻抗不平衡，方法：按下 **CH1**（对应通道），输入改为 **50Ω**
2. **通电不亮**：原因：后面板接口漏电
3. **不能放置于焦点处**（少数几次没事）

## 🐍 Pycharm & Git

使用 **PyCharm** 进行 GitHub 的远程仓库提交时，若出现图示的一直推送中，检查以下几点：

![PyCharm Git 推送失败]({attach}lab-images/P2.png)
1. 是否在 PyCharm 中登录了 GitHub 账户——**登录是每次都更新的**
2. 若不可行，**全局代理更新 git push 指令的代理路径**，使用 **socks5**
3. 更换 **http** 与 **socks5** 协议试一试
4. **买好点的代理服务器**，免费翻墙或低价翻墙易被封禁
5. GitHub 的 **SSH** 密钥是否生成
6. **如非必要，不使用上述服务**，**http** 更稳定高效（且无需密码）

## 📟 示波器

1. 仅有一个波形、信号无变化时：光电传感器被遮挡、透光光纤与延时板光电管未对准
2. 出现**刺猬波形** → 电触发盖过光触发，
3. 出现下图信号后，**光电传感器接触不良或延时板元件烧毁**

![异常信号波形]({attach}lab-images/P3.png)
4. 阶梯型脉冲，首尾相连，原因：阻抗不平衡，方法：按下CH1（对应通道），输入改为50Ω

![阶梯型脉冲波形]({attach}lab-images/P4.png)
5. 保存数据时，如无法返回保存界面，按照对应操作处理

![保存操作步骤1]({attach}lab-images/P5-1.png)
![保存操作步骤2]({attach}lab-images/P5-2.png)
6. 用示波器**捕获瞬时信号**时，触发电平要比要测的电平**稍微低一点**，不能低太多，避免噪声误触发


## 😄 科研笑话（真实经历）

2. **光电探测器没插电**
3. 示波器应该为 **us**，但误判成 **ms**
